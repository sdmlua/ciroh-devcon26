## CIROH DevCon 2026 — Workshop Materials

This repository contains all materials for the **CIROH Developers Conference 2026** workshop on Flood Inundation Mapping (FIM), delivered by the [Surface Dynamics Modeling Lab (SDML)](https://sdml.ua.edu/) at the University of Alabama.

The workshop covers two complementary tools built on the [FIMserv](https://github.com/sdmlua/FIMserv) framework:

| Module | Description |
|---|---|
| [Surrogate Modeling](surrogate-modeling/) | Coupled terrain-based OWP HAND FIM with HEC-RAS based FIM assimilated hybrid surrogate model (SM) to enhance LF-FIMs |
| [FIM Evaluation (FIMeval)](fimeval/) | Large-scale automated framework for benchmarking FIM accuracy |

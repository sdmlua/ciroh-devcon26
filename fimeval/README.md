## Large-Scale Flood Inundation Mapping (FIM) Evaluation Framework
The FIM Evaluation (FIMeval) framework provides a scalable, automated approach for benchmarking and validating Flood Inundation Maps across large spatial extents. It compares operational Low-Fidelity Physics-Based (LFPB) FIMs generated from the NOAA OWP HAND model against higher-fidelity reference inundation products, computing standard accuracy metrics (CSI, FAR, POD, F1) at reach and watershed scales to quantify model performance.
